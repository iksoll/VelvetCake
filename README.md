# VelvetCakes

Авторская кондитерская онлайн — заказывайте нежные, сочные и красивые торты и чизкейки на любой вкус.  
Проект разработан как курсовая работа с полным стеком: фронтенд + бэкенд + база данных.

---

## Возможности

- Просмотр каталога готовых десертов
- Создание индивидуального торта через **конструктор** (выбор бисквита, начинки, пожеланий к дизайну)
- Авторизация и регистрация пользователей
- Ролевая система:
  - **Пользователь** — оформление заказов, просмотр истории
  - **Менеджер** — управление заказами, статусами, каталогом
  - **Кондитер** — просмотр заказов, обновление статусов
- Админка для управления:
  - Товарами
  - Компонентами (бисквиты, начинки)
- Уведомления о смене статуса заказа
- Корзина и оформление заказа

## Технологии

### Фронтенд
- HTML5, CSS3, JavaScript
- Адаптивная верстка

### Бэкенд
- C#, ASP.NET Core Web API
- Entity Framework Core (Database-First подход)
- JWT-авторизация
- Swagger/OpenAPI для документации API
- Microsoft SQL Server

### Инфраструктура
- База данных: `VelvetCakes` (SQL Server)

---

## Запуск проекта

### 1. База данных
1. Подключитесь к SQL Server.
2. Выполните скрипт создания БД из `database/VelvetCakes.sql`.

### 2. Бэкенд
1. Откройте решение `VelvetCakes.sln` в Visual Studio.
2. Обновите строку подключения БД в `appsettings.json` и `ApplicationDbContext.cs` на вашу.
3. Запустите проект

### 3. Фронтенд
1. Откройте проект `Cake_Site`
2. Запустите файл `index.html`
